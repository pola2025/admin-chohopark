2

사이트 소유권 확인

URL 접두어로 등록하면 소유권 확인 화면이 나타납니다. 아래 방법 중 하나를 선택하세요.

🏆 방법 1: HTML 태그 (가장 쉬움)

1.

"HTML 태그" 옵션을 클릭합니다

2.

표시되는 메타 태그를 복사합니다

예시 형식:

<meta name="google-site-verification" content="xxxxxxxx" />
3.

복사한 메타 태그를 저에게 알려주세요

4.

웹사이트 <head> 섹션에 추가 후 배포합니다

5.

"확인" 버튼을 클릭합니다

📊 방법 2: Google Analytics (GA4 설치됨)

1.

"Google 애널리틱스" 옵션을 클릭합니다

2.

GA4가 이미 설치되어 있으면 자동으로 확인됩니다

※ 이미 GA4 (G-P63CJMK6LB)가 설치되어 있으므로 이 방법이 가장 빠를 수 있습니다

📄 방법 3: HTML 파일 업로드

1. 제공되는 HTML 파일을 다운로드

2. 웹사이트 루트 디렉토리에 업로드

3. 배포 후 확인 버튼 클릭

3

사이트맵 제출

📌 사이트맵 제출 방법

1.

왼쪽 메뉴에서 색인 생성 → Sitemaps 클릭

2.

"새 사이트맵 추가" 입력란에 아래 URL을 입력합니다:

sitemap.xml
3.

"제출" 버튼을 클릭합니다

🔗 사이트맵 URL

https://chorigol.net/sitemap.xml
→ 사이트맵 확인하기 (새 탭)

✅ 제출 후 상태 확인

• 성공: 초록색 체크 표시

• 대기 중: 처리에 며칠 소요될 수 있음

• 오류: 오류 메시지 확인 후 수정

4

색인 생성 요청 (선택사항)

중요한 페이지를 빠르게 Google에 등록하려면 개별 URL 색인 요청을 할 수 있습니다.

📌 색인 요청 방법

1.

Search Console 상단 검색창에 URL 입력

2.

Enter 눌러 URL 검사 실행

3.

"색인 생성 요청" 버튼 클릭

🎯 색인 요청 추천 URL

https://chorigol.net
https://chorigol.net/booking
https://chorigol.net/about

※ 하루 색인 요청 횟수에 제한이 있습니다 (약 10-20회)

💡 알아두면 좋은 정보

•

색인 생성 시간: 사이트맵 제출 후 Google이 크롤링하는 데 며칠에서 몇 주가 소요될 수 있습니다

•

검색 결과 반영: 색인 생성 후 실제 검색 결과에 나타나기까지 추가 시간이 필요합니다

•

정기 확인: Search Console에서 오류, 경고 메시지를 주기적으로 확인하세요

•

사이트맵 자동 업데이트: Next.js의 sitemap.ts가 자동으로 최신 상태를 유지합니다
