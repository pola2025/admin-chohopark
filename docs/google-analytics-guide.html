Google Analytics 접속 및 계정 생성

아래 버튼을 클릭하여 Google Analytics에 접속하세요:

Google Analytics 열기
▸

Google 계정으로 로그인합니다 (Gmail 계정 사용)

▸

"측정 시작" 버튼을 클릭합니다

▸

이미 계정이 있다면 왼쪽 하단 ⚙️ 관리 → + 계정 만들기

2

계정 및 속성 설정

⚠️ 아래 정보를 그대로 입력하시면 됩니다

📌 계정 설정

계정 이름: 초호쉼터

※ 데이터 공유 설정은 기본값 그대로 두고 "다음" 클릭

📌 속성 설정

속성 이름: 초호쉼터 홈페이지
시간대: 대한민국 (GMT+09:00)
통화: 대한민국 원 (₩)

※ "다음" 클릭

📌 비즈니스 세부정보

업종: 여행 (Travel)
규모: 소규모 (1~10명)

※ "다음" 클릭

📌 비즈니스 목표

아래 항목들을 선택하세요:

※ "만들기" 클릭 후 약관에 동의

3

데이터 스트림 생성 및 측정 ID 복사

📌 데이터 수집 시작

1.

"데이터 수집 시작" 화면에서 "웹"을 선택합니다

2.

웹사이트 URL을 입력합니다:

https://chohopark.com
3.

스트림 이름을 입력합니다:

초호쉼터 웹사이트
4.

"향상된 측정"이 켜져 있는지 확인 (기본 켜짐)

5.

"스트림 만들기" 버튼 클릭

🎯 측정 ID 복사하기

스트림이 생성되면 "측정 ID"가 표시됩니다.
G-로 시작하는 코드입니다 (예: G-ABC123XYZ)

측정 ID 위치:

웹 스트림 세부정보 → 측정 ID 옆 복사 아이콘 클릭

💡 나중에 측정 ID 찾는 방법

관리(⚙️) → 데이터 수집 및 수정 → 데이터 스트림 → 스트림 클릭 → 측정 ID 확인

4

측정 ID 입력

복사한 측정 ID를 아래에 입력해주세요:

G-로 시작하는 측정 ID를 입력하세요

✅ 완료 후 알려주세요!

측정 ID를 입력하시면 홈페이지에 Google Analytics를 연동하고,
관리자 대시보드에서 방문자 통계를 확인할 수 있게 설정해드립니다.
